# bilibili

### 安装依赖
```
yarn install
```

### 项目运行 
```
yarn serve
```

### 项目打包

```
yarn build
```

##  项目描述

本项目用到的技术栈 vue + vue-router + ES6/7 + less + flex + vant + axios + vw移动端适配

本项目参照  B站 https://www.bilibili.com/video/BV1vT4y137So

项目预览地址

http://47.94.159.197/

完成功能

* 登录注册
* 权限验证
* 编辑个人信息
* 上传头像
* 下拉加载
* 视频播放
* 收藏文章
* 关注用户
* 盖楼评论
* 发表评论
* 筛选分类
